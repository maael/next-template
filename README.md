# Next Template

A simple repo to act as a template for a Next.js project.

Includes:

- Next.js
- TypeScript
- Prettier
- Eslint
- Prettier + Eslint on commit
- Fathom Analytics hook
- next-seo
- Sane CSS reset/starting point

## Usage

- Remove [GitHub workflow](./github/workflows/usage.yml) to track usage.
